# Void
A simulation of an operating system.  
This is the first version, so its not finished!  

Funcionalidades  
-Aplicativos básicos como calculadora, chatbot(Órion), bloco de notas,lista de tarefas, prompt de comandos, entre outros...  
-Interfáce de texto de fácil entendimento.  

Como usar  
-Assim que o sistema for iniciado, ele pedirá um usuário e uma senha, segue abaixo:  
Usuário: void  
Senha: void  
-Espere aparecer a opção de apps, e escolha o aplicativo desejado.
